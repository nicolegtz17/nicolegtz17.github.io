# nicolegtz17.github.io

Polished portfolio built with Vite, React, and TypeScript.

## Local development

```bash
npm install
npm run dev
```

## Build + preview

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

The deployment is handled by GitHub Actions. Push to `main` and the workflow
builds the site and deploys `dist/` using the official Pages actions.
In GitHub, make sure Pages is set to use GitHub Actions as the source.

## Resume file

Place your real resume at `public/resume.pdf` to replace the placeholder.
